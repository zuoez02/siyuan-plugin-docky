# 侧边栏工具

将你的块添加到侧边栏的工具

## 使用方法

1. 安装插件并启用
2. 右上角出现侧边栏工具的 Dock
3. 复制你想固定的块的 id，点击侧边栏工具右上角的+
4. 可以在这里调整顺序、删除、绑定到 dock 栏中
